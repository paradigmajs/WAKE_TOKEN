# Audit Scope

The audit scope includes all smart contracts located in the contracts directory.

Primary contracts include:

WAKEToken

WakeBeneficiaryVestingVault

WakeControlledEmissionVault

WakeCustodyVault

WakeLiquidityVault

WakePresaleMerkleVesting

Additional vault contracts handling token allocation.

## Audit Focus

The audit should evaluate:

- token supply integrity
- access control logic
- vesting implementation
- claim validation
- arithmetic safety
- ownership permissions
- token release mechanisms
